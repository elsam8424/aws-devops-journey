# AWS DevOps Journey

Repositorio público de aprendizaje AWS desde cero hasta nivel senior DevOps. Documenta labs prácticos reales con **Floci** (emulador AWS local), **AWS CLI** y **Terraform**.

Objetivo: construir un portfolio verificable de competencias cloud — no teoría aislada, sino comandos, configuraciones y decisiones documentadas paso a paso.

## Stack

| Herramienta | Uso |
|---|---|
| [Floci](https://github.com/floci-io/floci#why-floci) | Emulador AWS local (`localhost:4566`) |
| AWS CLI | Gestión de recursos y automatización |
| Python / Bash | Scripts y utilidades |
| Terraform | Infraestructura como código *(próximamente)* |
| AWS Free Tier | Validación en entorno real |
